# Script/Voice Gen

Script/Voice Gen is a Python program which generates a script using the chatGPT API, and generates audio files from those characters using the FakeYou API.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

### To use, run

```bash
python main.py
```
